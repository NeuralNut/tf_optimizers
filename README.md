Implementation of optimizers in tensorflow from scratch. See `optimizers.py`.

Optimizers
- Gradient descent
- Gradient descent with gradient clipping
- Momentum
- Nesterov momentum
- Adagrad
- Adadelta
- RMSProp
- Adam
- Adamax
- SMORMS3

Might be useful.
